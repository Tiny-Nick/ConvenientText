ClassIsland 的文本编辑

桌面一个 ✎ 按钮，点开就能改主界面上的字、颜色和字号。

## 怎么用

- 装好插件后，在设置 → 组件里把「便捷文本」拖到主界面上
- 桌面上会出现一个灰色圆按钮，点一下弹出编辑框
- 改文字、选颜色、调字号，点确定，主界面立刻变

## 兼容性

- ClassIsland 2.1.0.0
- Windows 10 / 11（理论上 Linux/macOS 也能跑，没测过）
