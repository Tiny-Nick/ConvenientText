# ClassIsland 便捷文本插件 1.2.0.0

## 插件简介

用传统 CI 文本组件想在主界面上改一行字，得打开设置 → 找到组件 → 点开组件设置 → 改完再关掉……改一个字要翻好几层。

**便捷文本**由此孕育而生：它在你的**桌面**上放一颗置底的小悬浮按钮（不挡课、不抢焦点、可以随便拖到顺手的位置），点一下就可以快速更改

### 🧩 多文本组件

- 主界面最多可添加 **5 个**"便捷文本"组件，每个组件独立显示一行文字
- 每个组件有自己的**标识圆点**，一眼分清谁是谁
- 文字**颜色、字号**自由调整

### ✏️ 便捷编辑窗口

- 点击主界面上的组件文字，或从悬浮按钮进入，都能打开编辑窗口
- 文字、颜色、字号一个窗口全搞定，全套**亚克力无边框**设计（作者还是那么的喜欢亚克力）

### 📋 全局预设库

- 常用文案提前存进预设库，编辑时一键加载，不用每次重打，预设随时增删，全组件共享

### ⏰ 时间段自动控制

- 给每个组件单独设置显示时间段（支持跨午夜，如 22:00 ~ 次日 6:00）
- 上课显示、放学隐藏，全自动，无需每天手动开关

### 🔌 U盘"防断头"提醒

- 检测到U盘插入时，弹出温馨提醒：**"请注意黑板的推拉，以防止U盘和白板插口的损坏"**
- 拯救每一根插在白板上的U盘（有没有人能替作者联系一下 ⌜路易十六⌟）
